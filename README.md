# Auto React Bot (14 Bots)

This project runs 14 Telegram bots simultaneously with fixed emoji reactions.

## Features

- 14 bots in single script
- Fixed emoji per bot
- Ultra fast auto reaction
- No JSON required
- Deployable on Railway, Render, Fly.io

## Environment Variables

Set these secrets:

BOT_1_TOKEN=
BOT_2_TOKEN=
BOT_3_TOKEN=
...
BOT_14_TOKEN=

## Deploy

Supports:

- Railway
- Render
- Fly.io
- VPS
